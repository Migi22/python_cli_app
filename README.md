# Python CLI Utility Toolkit

A menu-driven Python CLI app that includes:

- ✅ To-Do List
- ✅ Unit Converter
- ✅ Mini Quiz Game
- ✅ Password Generator
- ✅ Alarm Clock

## Getting Started

Run the main program with:

```bash
python main.py
```
